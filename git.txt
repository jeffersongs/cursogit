---------------------------
File Status Lifecycle
untracked - não marcado, quando colocamos no diretorio mas ainda não colocamos no git (não tem versão ainda)
unmodified - não foi modificado, está no git e não teve nenhuma modificação
modified - modificado, quando vc edita e modifica algo
staged - é uma area para criar uma versão, é como se fosse um repositorio para quando tudo que for mexido, criarmos uma versão nova

------------------------------------
git status - mostra como está o repositório
git add nomedoarquivo.ext - adiciona o arquivo pra area staged
